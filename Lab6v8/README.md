# Лабораторна робота №6. Варіант №8

**Тема:** *Лямбда-вирази, анонімні функції та делегати у C#.*
---
**Мета:** 

- Закріпити знання про делегати та події.
- Навчитись використовувати анонімні методи, лямбда-вирази і вбудовані делегати (Func<>, Action<>, Predicate<>).
- Отримати практичний досвід застосування лямбда-виразів у колекціях і LINQ-операціях.

---

## Опис виконаної роботи (Варіант №8)
- Створив колекції чисел.
- Реалізував Func<int,int,bool> для перевірки простого числа.
- Використання Action<List<int>> для друку списку.
- Використав Predicate<int> для видалення елементів
- Використав функцію Where для фільтрації простих чисел
- Реалізував видалення зайвих елементів через RemoveAll + Predicate
- Реалізував добуток усіх залишених чисел через LINQ Aggregate

## Висновки:
У процесі виконання лабораторної роботи було: 
- практично реалізовано логічний делегат Func<int,int,bool> для перевірки простих чисел у колекції List<int>;

- застосовано делегат Action<List<int>> для друку елементів у консоль;

- використано Predicate<int> для визначення й видалення непотрібних значень;

- здійснено обробку та аналіз даних за допомогою бібліотеки LINQ, включно з фільтрацією, проєкцією та агрегуванням;

- продемонстровано принцип комбінування методів у ланцюг виконання, що є ключовою властивістю multicast-делегатів у C# середовищі розробки.

Робота підтверджує, що делегати та lambda-вирази є:
- зручним способом передавати методи як аргументи,
- ефективним інструментом для обробки колекцій без - створення зайвих іменованих функцій,
- читабельним механізмом для логічних перевірок і подійної взаємодії.
